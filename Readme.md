# Modern SaaS CRM Dashboard

A modern, responsive CRM dashboard built as part of the Dale Edge Industry Internship Program 2026 assessment. The application simulates a production-style SaaS dashboard using frontend technologies and mock data.

## Features

### Login

* Email and password validation
* Password visibility toggle
* Remember Me option
* Responsive design

### Dashboard

* KPI cards
* Revenue chart
* Customer statistics
* Recent activity feed

### Customers

* Search customers
* Sort customer list
* Filter customers
* Pagination

### Orders

* Search orders
* Filter by status
* Edit order status
* Delete orders

### Settings

* Update profile information
* Form validation
* Light/Dark theme
* Save and Reset settings

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Chart.js
* Local Storage

## Installation

Clone the repository:

```bash
git clone <https://github.com/zavithibrahim4-art/intership-SaaS-project>
```

Open the project folder and launch `index.html` using Live Server or any local web server.

## Assumptions

* Authentication is simulated.
* Customer and order data use static JSON.
* No backend or database is included.

## Known Limitations

* No real user authentication.
* Data is not permanently stored.
* Dashboard data is for demonstration purposes.

## Technical Decision

I chose to build this project using HTML, CSS, and JavaScript instead of a framework to demonstrate a strong understanding of core frontend concepts while keeping the application lightweight and easy to maintain.
